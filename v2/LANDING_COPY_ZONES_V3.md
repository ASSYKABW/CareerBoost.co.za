# Landing Copy Zones (Phase-2 Writing Pass)

This file marks the minimal copy zones to edit next, without touching layout architecture.

## Priority 1: Above-the-fold

- `Hero pill`: short context label only
- `Hero headline`: one strong outcome promise
- `Hero lead`: one sentence max
- `Hero primary CTA`: action verb + low friction
- `Hero secondary CTA`: orientation action (how it works/demo)

## Priority 2: Mid-funnel narrative

- `Problem title` + 3 bullets
- `Solution title` + 3 bullets
- `Features section heading`
- `Feature card one-liners` (keep to one sentence each)

## Priority 3: Decision support

- `Pipeline heading`
- `Pipeline stage card body copy` (short, action-led)
- `Testimonials quotes` (1–2 lines each)
- `Pricing headings + bullets` (value clarity)
- `FAQ question wording` (high-friction concerns first)

## Priority 4: Conversion close

- `Final CTA heading`
- `Final CTA support line`
- `Final CTA button label`

## Constraints for Copy Pass

- Keep section structure unchanged (`welcome.route.js` layout wrappers stay fixed)
- Avoid adding extra paragraphs
- Keep mobile density low (no long blocks)
- Preserve CTA consistency (`Start free` primary action)
